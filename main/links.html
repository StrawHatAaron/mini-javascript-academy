Further Reading

After getting started with Javascript Academy, we suggest you check out these links and resources to continue improving:

E-books

  • Eloquent Javascript is a free e-book that teaches Javascript at an intermediate to advanced level. It's a great resource for understanding abstract programming concepts. Explanations are clear, detailed, and easy to follow.

References

  • Mozilla's JavaScript Guide and W3Schools provide useful guides, references, and tutorials for all levels. Both have definitions for all the fundamental terms, W3School also has "try-it-yourself" examples alongside definitions. It should be noted that only learning from unrelated tutorials is not an efficient way to learn. Resources such as these should always be used as a supplement for actual lessons.
